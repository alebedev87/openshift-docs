= Installing AWS Load Balancer Operator on a Security Token Service cluster
include::_attributes/common-attributes.adoc[]
:context: albo-sts-cluster

toc::[]

You can install the AWS Load Balancer Operator on a Security Token Service (STS) cluster.

The AWS Load Balancer Operator relies on `CredentialsRequest` to bootstrap the Operator and for the `AWSLoadBalancerController` instance. The AWS Load Balancer Operator waits until the secrets provioned by the Cloud Credential Oerator are created and available.
